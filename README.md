# LnQ WhatsApp Bot

This directory is intended for a WhatsApp bot integration for the LnQ project.

## Purpose

The bot will serve as an interface for interacting with the LnQ system via WhatsApp, allowing users to perform actions such as:

-   Receiving order notifications.
-   Checking order statuses.
-   Querying product information.
